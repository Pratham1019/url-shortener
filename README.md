# url-shortener
Simple, fast, and scalable URL shortener using FastAPI and Redis.

* Generate short, shareable URLs instantly.
* Redis-backed storage for fast lookups and expiration handling.
* Configurable link lifetimes with automatic expiry.
* Clean REST API built with FastAPI.
* Lightweight frontend built with vanilla HTML, CSS, and JavaScript.
* Deployed on Render with Upstash Redis.
* Deployed the server and site on Render.
